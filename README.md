# Housing-Market-Dashboard
## Project Overview
This project focuses on analyzing housing sales data using an interactive Tableau dashboard embedded into a Flask web application.

## Technologies Used
- Python
- Flask
- Tableau Public
- HTML/CSS

## Features
- Interactive dashboard visualization
- Analysis of housing prices
- Web-based access using Flask

## How to Run the Project
1. Install Flask
2. Run app.py
3. Open http://127.0.0.1:5000/ in your browser
